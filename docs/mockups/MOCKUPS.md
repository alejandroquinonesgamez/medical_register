# Mockups y Diagramas

Esta sección contiene los mockups y diagramas del proyecto.

## Diagrama de Flujo del Manual de Usuario

Diagrama de flujo que muestra el proceso de uso de la aplicación, desde el registro inicial hasta la consulta de estadísticas.

![Diagrama de Flujo del Manual de Usuario](user-manual.png)

### Descripción

El diagrama muestra el flujo completo de la aplicación:

1. **Inicio**: Al acceder por primera vez, se requiere registro de datos
2. **Registro de Datos**: Nombre, apellidos, fecha de nacimiento y talla
3. **Almacenamiento**: Los datos se guardan en localStorage del navegador
4. **Pantalla Principal**: Una vez registrados, se accede a la pantalla principal
5. **Registro de Peso**: Permite registrar peso (una vez al día)
6. **Cálculo del IMC**: Se calcula automáticamente el Índice de Masa Corporal
7. **Guardar Registro**: Se guarda el registro con fecha y hora
8. **Actualizar Histórico**: Se actualiza el histórico de registros
9. **Consultar Estadísticas**: Se pueden ver estadísticas como número total de pesajes, peso máximo y mínimo
